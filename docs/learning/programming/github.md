## Overview

GitHub is the online platform we use to host our code repositories. On top of
plain Git, it adds tools for collaboration: pull requests, issues, and project
boards. This lesson gets your account ready so you can join the team's work.
Issues and project boards are covered in detail in the **Kanban & Agile** lesson.

## Prerequisites

- A school or personal email address.

## What you'll learn

- What GitHub is and how it relates to Git.
- How to set up and secure a GitHub account.
- How to present yourself to the team with a profile README.

## Steps & acceptance criteria

- [ ] Create a GitHub account if you don't already have one. (If your team uses a
      GitHub organization, ask a mentor to add you — but you'll do the learning
      modules in your **own** repositories.)
- [ ] Set a profile picture for your account.
- [ ] Enable **two-factor authentication (2FA)** to keep your account secure.
- [ ] Create a custom profile README that introduces you to the team, and share
      the link with a mentor. (A profile README lives in a repository named exactly
      after your username.)
